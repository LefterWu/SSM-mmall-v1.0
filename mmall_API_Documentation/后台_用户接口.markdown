#### 1.后台管理员登录

**/manage/user/login.do**


> rerequest

```
String username
String password
```

> reresponse

success

```json
{
    "status": 0,
    "msg": "登录成功",
    "data":{
        "id": 1,
        "username": "admin",
        "password": "",
        "email": "admin01@mmall.com",
        "phone": "18510001001",
        "question": "新问题",
        "answer": "新答案",
        "role": 1,
        "createTime": 1478422605000,
        "updateTime": 1543238043000
    }
}
```

fail
```json
{
    "status": 1,
    "msg": "用户名或密码错误"
}
```

```json
{
    "status": 1,
    "msg": "不是管理员账号，无法登录"
}
```
